# 🏠 Valuation & Feature Engineering for Real Estate Market

## 📌 Descripción del Proyecto
Este repositorio contiene el análisis exploratorio de datos (EDA), la ingeniería de características (*Feature Engineering*) y la construcción de un modelo predictivo basado en **XGBoost** para la estimación de precios en el mercado inmobiliario. 

El modelo alcanza un desempeño de **MAE de 9.94M JPY** y un **$R^2$ de 0.8029** sobre el conjunto de prueba temporal, garantizando la ausencia de *data leakage* y optimizando la interpretabilidad de negocio mediante métricas de ganancia (*Gain*).

---

## 🛠️ Estructura del Repositorio
* `notebooks/`: Cuaderno Jupyter con la metodología completa (Ingesta, Limpieza, EDA, Modelado e Interpretación).
* `data/`: Estructura de datos procesada y datasets intermedios.
* `src/`: Scripts auxiliares de pipeline y utilidades.
